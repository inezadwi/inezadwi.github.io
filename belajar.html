Apa Itu Website?

Website adalah serangkaian halaman web berisi informasi yang terhubung satu sama lain dan diakses melalui internet. Pada era digital saat ini, website telah menjadi salah satu elemen penting di dalam kehidupan manusia

Apa Manfaat Website?

Selain media dalam menyampaikan informasi, website berfungsi sebagai platform komunikasi, menarik dibaca pengguna, untuk hiburan, dan bahkan tempat transaksi jual beli online

Jenis Website

Berdasarkan Sifat

  1. Website dinamis berisi konten yang selalu berubah setiap saat. Contohnya website toko online dan website internet banking
  2. Website statis. Kontennya sangat jarang diubah. Contohnya, website landing page.

Berdasarkan Tujuan

  1. Personal website berisi informasi pribadi seseorang.
  2. Corporate web dimiliki oleh perusahaan.
  3. Portal website memiliki banyak layanan berita, email, dan jasa-jasa lain.
  4. Website media sharing berbagi media antarpengguna seperti gambar, video, musik. Contohnya: Flickr, Youtube, Soundcloud
  5. Forum website digunakan sebagai sarana diskusi pengunjungnya
